lobby_info​

Info Updates​

keyCategoryValuesNotesSince Version
queueIDlobby_infoThe queueID of the currently chosen game mode. See notes below126.0
player_infolobby_infoInformation about the local player See notes below240.0

queueID notes​

The queueID of the currently chosen game mode. Example:

{"feature":"lobby_info","category":"lobby_info","key":"queueId","value":1090}

  • You can find here a list of possible queue ids.
  • 1090 is the queueID of TFT regular match.
  • 1100 is the queueID of TFT ranked match.

player_info note​

List of parameters:

  • displayName - the player's display name
  • gameName - the player's game name
  • puuid - the player's unique ID
  • summonerId - the summoner ID
  • tagLine - the player's tagline

Data Example:

{"feature":"lobby_info","category":"lobby_info","key":"player_info","value":"{\"displayName\":\"TacoNano888\",\"gameName\":\"TacoNano888\",\"puuid\":\"06c5c9b8-3096-50fa-bd09-d07cc66a121e\",\"summonerId\":3106004487710176,\"tagLine\":\"12345\"}"}
